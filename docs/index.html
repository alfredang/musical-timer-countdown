Getting Started

The Musical Class Break Timer is a simple yet powerful countdown timer designed for classrooms, study sessions, and productivity breaks. No installation required - just open and start timing!

Preset Times

Quick buttons for 15m, 30m, 45m, 1h, 1h 15m, and 1h 30m

Custom Input

Enter any time in hours, minutes, and seconds

Visual Progress

Animated ring shows remaining time at a glance

Custom Sounds

Upload your own MP3 or use built-in alarm sounds

Setting a Timer

Method 1: Use Preset Buttons

Click any of the preset buttons below the timer display:

  • Click a preset time button (e.g., "15m" for 15 minutes)
  • The timer will be set automatically
  • Click Start to begin the countdown

Method 2: Manual Entry

Enter a custom time using the input fields:

  • Click on the HR, MIN, or SEC input field
  • Type your desired time (values auto-adjust if you enter more than 59 for minutes/seconds)
  • Click Start to begin the countdown

Timer Controls

Start / Resume

Begins the countdown. Changes to "Resume" when paused.

Pause

Temporarily stops the timer. Click Resume to continue.

Reset

Clears the timer completely back to 00:00:00.

Stop Alarm

Appears when timer ends. Silences the alarm and resets.

When the Timer Completes

When your countdown reaches zero:

  • The display shows "Time's Up!" with a flashing animation
  • Your selected alarm sound plays continuously on loop
  • A red Stop Alarm button appears
  • Click the button to silence the alarm and reset the timer
Tip: The alarm will continue playing until you click Stop Alarm, so you won't miss it even if you step away!

Sound Settings

Volume Control

Use the volume slider at the bottom of the screen to adjust the alarm volume. Click the speaker icon to mute/unmute.

Choosing an Alarm Sound

  1. Click the gear icon in the top-right corner to open Settings
  2. In the "Alarm Sound" section, choose from: Dream, Inspire, Wing, or Custom
  3. Click Preview Sound to hear your selection
  4. Your choice is saved automatically

Uploading Custom Audio

  1. Open Settings and select the Custom sound option
  2. Click Upload MP3 to choose a file from your device
  3. Select an MP3 file (maximum 5MB)
  4. The filename will appear next to the button
  5. Click Preview Sound to test your upload
Note: Custom audio is stored in your browser's localStorage. If you clear your browser data, you'll need to upload it again.

Tips & Tricks

Keyboard Shortcuts

While focused on an input field, use Tab to move between hours, minutes, and seconds fields.

Browser Tab Title

The browser tab shows the current countdown time, so you can see the remaining time even when the timer is in a background tab!

Mobile Friendly

The timer is fully responsive and works great on phones and tablets. Just open it in your mobile browser!

Troubleshooting

Sound not playing?

  • Check that the volume slider is not at zero
  • Ensure the mute button (speaker icon) is not enabled
  • Some browsers require user interaction before playing audio - try clicking anywhere on the page first

Custom audio not working?

  • Ensure the file is a valid MP3 format
  • Check that the file size is under 5MB
  • Try re-uploading the file

Settings not saving?

  • Make sure your browser allows localStorage
  • Private/Incognito mode may not persist settings
  • Clearing browser data will reset all settings
